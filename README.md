# ALTIX- Enciclopedia _Pokemon

# 📊 Proyecto: Enciclopedia Pokémon
**Desarrollado por ALTIX**

Este repositorio contiene el diseño de base de datos, procesamiento de información y visualizaciones de la Enciclopedia Pokémon.

## 📁 Estructura del Repositorio

El proyecto está organizado bajo estándares profesionales de análisis de datos:

* **`/datos`**: Contiene los datasets originales (`.csv`), los archivos normalizados y los scripts de limpieza de datos utilizando Pandas.
* **`/sql`**: Incluye el script de creación de la base de datos relacional y los scripts de inserción automatizada.
* **`/graficas`**: Contiene el código fuente que genera las visualizaciones del proyecto y el documento `ANALISIS_GRAFICAS.md` con la interpretación de negocio.
* **`/documentos`**: Contiene la presentación corporativa oficial de ALTIX con la definición de roles del equipo, el documento detallado de análisis e interpretación de las gráficas, y el Diagrama Entidad-Relación (ER) de la base de datos.

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Base de Datos:** MySQL
* **Lenguaje Principal:** Python
* **Análisis y Manipulación:** Pandas, NumPy
* **Visualización:** Matplotlib
* **Conexión a BD:** mysql-connector-python, SQLAlchemy

1. Clona este repositorio en tu máquina local.
2. Asegúrate de tener Python instalado y crea un entorno virtual (opcional pero recomendado).
